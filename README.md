# 🐾 Standard Feline Calendar (SFC)

> **"Time is relative, but for a cat, it's absolute."**

**标准猫历 (SFC)** 是一套将人类线性时间重构为“猫本位”的开源授时系统。它不仅改变了我们对时间的计量方式，更通过生理与本能维度，重新定义了猫咪眼中的生命律动。

## 🌟 核心特性

- **1:5 时间膨胀**：1 地球年 = 5 猫历年。
- **地理位置感知**：全球边缘节点自动适配时区，提供毫秒级授时。
- **行为学预报**：实时计算“跑酷概率”、“睡眠欲”及“掉毛率”。
- **极客友好**：提供 RESTful API 与精美的实时监控面板。

## ⏳ 历法标准 (The Standards)

- **纪元 (Epoch)**: 公元前 7500 年（人类服侍猫咪的考古证据元年）。
- **猫时 (Feline Hours)**: 将 24 小时划分为 6 个周期：
  - `伏击时 (Ambush)` | `晨猎时 (Dawn Hunt)` | `巡窗时 (Window Patrol)`
  - `大梦时 (Great Dream)` | `黄昏猎 (Twilight Hunt)` | `理毛时 (Grooming)`
